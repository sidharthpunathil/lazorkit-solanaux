# Documentation

This directory contains the documentation for Lazorkit Solana UX Examples.

## Structure

- `getting-started.md` - Quick start guide
- `tutorials/` - Step-by-step tutorials
- `api/` - API reference
- `examples/` - Code examples

## Setup

Documentation is built using [Fumadocs](https://www.fumadocs.dev/), a beautiful documentation framework for Next.js.

To view the documentation locally:

```bash
bun run dev
```

Then navigate to `/docs` in your browser.
